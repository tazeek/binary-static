Binary.com Open-Source Projekte

Binary.com ist ein aktiver Vertreter der Open-Source-Bewegung und ist dabei, eine Anzahl an Komponenten seiner Codebase offen zur Verfügung zu stellen.

Statischer Inhalt

Verbessern Sie den Front-End Inhalt von Binary.com

Der statische Inhalt (CSS, Javascript) der Binary.com Website steht als Binary-Static Open Source Projekt auf github zur Verfügung. Möchten Sie der Binary.com Website einen neuen Skin geben, die Browserkompatibiltät, die Seitenaufrufgeschwindigkeit oder die Javascript Leistung erhöhen? Dann verwenden Sie bitte das Projekt und übersenden Sie uns Pull-Requests für Ihre vorgeschlagenen Programmänderungen.

Perl Module

Die CPAN Beiträge von Binary.com

Binary.com ist ein aktives Mitglied der Perl Community und ist dabei, eine Anzahl seiner internen Dienstprogramm-Module auf CPAN hochzuladen. Wenn Sie ein Pearl-Fan sind, können Sie beitragen, indem Sie diese Module bewerten und Pull-Anfragen bei github einreichen.
